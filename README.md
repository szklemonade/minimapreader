# 概要 / Overview
ブラウザ上にミニマップを表示する
Displays a minimap on the browser.

文字、画像などを分類して％表示する
Categorizes and displays the percentage of elements such as text and images.

それぞれの表示は設定から変更可能
Each display option can be customized from the settings.

# インストール方法 / Installation
このリポジトリをダウンロードまたはクローンします。
Download or clone this repository.

Chromeの拡張機能ページ（chrome://extensions/）を開きます。
Open the Chrome extensions page (chrome://extensions/).

右上の「デベロッパーモード」を有効にします。
Enable "Developer mode" in the top right corner.

「パッケージ化されていない拡張機能を読み込む」をクリックし、ダウンロードしたフォルダを選択します。
Click "Load unpacked" and select the downloaded folder.

# 使い方 / Usage
設定方法 / Settings
拡張機能のアイコンをクリックし、設定画面から表示する要素（文字、画像など）の種類や表示方法を変更できます。
Click the extension icon and use the settings screen to change which elements (such as text or images) are displayed and how.
# ショートカットキー / Keyboard Shortcuts
Alt + M：ミニマップの表示/非表示
Alt + M: Show/Hide the minimap

Alt + Shift + C：ミニマップの折りたたみ
Alt + Shift + C: Collapse the minimap

Alt + Shift + L：凡例の表示
Alt + Shift + L: Show the legend
